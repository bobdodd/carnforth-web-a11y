Highlight.js Test Examples

This page contains various HTML code examples to test highlight.js integration in the Carnforth Web A11y extension.

Simple HTML Example

Basic HTML with standard elements and attributes. 
<div class="container">
  <h1>Hello World</h1>
  <p>This is a paragraph with <a href="https://example.com">a link</a> inside.</p>
</div>

Complex Nested HTML

More complex HTML with deep nesting. 
<div class="section" role="region" aria-label="User profile information">
  <header class="section__header">
    <h2 class="section__title">User Profile</h2>
    <div class="section__actions">
      <button type="button" class="btn btn--edit" aria-label="Edit profile">
        <svg class="icon"><use xlink:href="#edit-icon"></use></svg>
        Edit
      </button>
    </div>
  </header>
  <div class="section__content">
    <form id="profile-form" class="form">
      <div class="form__row">
        <label for="user-name" class="form__label">Name</label>
        <input id="user-name" type="text" class="form__input" 
               value="John Doe" required 
               aria-describedby="name-help">
        <small id="name-help" class="form__help">Enter your full name as it appears on official documents.</small>
      </div>
    </form>
  </div>
</div>

Accessibility-focused HTML

HTML with many ARIA attributes and roles. 
<div role="dialog" aria-labelledby="dialog-title" aria-describedby="dialog-desc" aria-modal="true" class="modal">
  <div class="modal__header">
    <h2 id="dialog-title" class="modal__title">Confirm Action</h2>
    <button type="button" class="modal__close" aria-label="Close dialog">×</button>
  </div>
  <div class="modal__body">
    <p id="dialog-desc">Are you sure you want to delete this item? This action cannot be undone.</p>
    <div class="form-field">
      <input type="checkbox" id="dont-show-again" aria-describedby="checkbox-help">
      <label for="dont-show-again">Don't show this again</label>
      <span id="checkbox-help" class="visually-hidden">If checked, this confirmation dialog will not be shown for future delete actions.</span>
    </div>
  </div>
  <div class="modal__footer">
    <button type="button" class="btn btn--secondary" aria-label="Cancel deletion">Cancel</button>
    <button type="button" class="btn btn--danger" aria-label="Confirm deletion">Delete</button>
  </div>
</div>
