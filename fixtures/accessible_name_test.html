Testing Instructions

  1. Open this page in Chrome
  2. Use the Carnforth Web A11y extension to run the accessible name touchpoint test
  3. Verify that issues are detected for elements missing accessible names:
    • Empty button (Test 1)
    • Image without alt text (Test 2)
    • Unlabeled input (Test 3)
    • Clickable div without role (Test 4)
    • Non-descriptive button text (Test 5)
    • Non-descriptive link text (Test 6)
    • Image with insufficient alt text (Test 7)
    • Input with placeholder only (Test 12)
    • Icon button without accessible name (Test 14)
    • Custom select without ARIA (Test 15)
    • Button with image missing alt (Test 19)
    • SVG without accessible name (Test 21)
    • Dialog without accessible name (Test 22)
  4. Verify that properly labeled elements (Tests 8-11, 13, 16-18, 20, 23) do not produce warnings or errors
